## 缔零法则 MCP
高精度内容安全风控产品

一款基于 Agentic AI 技术构建的高精度内容安全风控产品，让开发者在 10 分钟内保护您的 AI 应用程序。

### 使用配置

```json
{
    "lawgenesis-mcp": {
        "url": "https://mcp.lawgenesis.cn/api/sse",
        "headers": {
            "Authorization": "Bearer <your-token>"
        }
    }
}
```

### Token 获取方式

前往 [mcp.lawgenesis.cn](https://mcp.lawgenesis.cn/) 申请
